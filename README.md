Copercog

->É um sistema desenvolvido  com intenção de  profissionalizar   os serviços da cooperativa Copercog,  focado em pontos críticos de suas operações como cadastro de produtos,vendas ,fluxo de pedidos e outras tarefas,  visualmente as telas são intuitivas e fáceis de usar , e a usabilidade é totalmente centralizada no usuário com recursos de interação responsivas .

##Funcionalidades:
- Cadastro de produtos
- Atualização de produtos
- Exclusão de produtos
- Listagem de produtos
- Controle de estoque
- Cadastro de pedidos
- Atualização de status dos pedidos
- Exclusão de pedidos
- Controle de usuários


##Como Executar o Projeto:

1. Importe o script SQL no MySQL
2. Configure o banco de dados no projeto
3. Abra o projeto no NetBeans
4. Execute o projeto utilizando Maven
